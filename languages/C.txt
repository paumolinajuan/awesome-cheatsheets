PERSONALIZACIÓN DE PÁGINAS WEBS

1. HTML (Estructura)

<header> Pagina Web
<nav> [Ejemplo].com
<main> Del tema del que se hablara en la página

ATRIBUTOS ÚTILES
class: Para aplicar estilos específicos
id: Seleccionar elementos únicos

2. CSS (Estilos)
Selectores:
: Clase (ej. .mi-clase)
#: ID (ej. #mi-id)
elemento: Selector de etiqueta (ej. h1)

Propiedades comunes:
color: Cambia el color del texto.
background-color: Cambia el color de fondo.
font-size: Tamaño de la fuente.
margin: Espaciado exterior.
padding: Espaciado interior.

3. JavaScript (Interactividad)

Funciones básicas:
document.getElementById(): Selecciona un elemento por ID.
addEventListener(): Agrega eventos a los elementos.

function cambiarColor() {
    document.body.style.backgroundColor = "lightblue";
}

4. Herramientas y Frameworks

Bootstrap: Framework CSS para diseño responsivo.
  Clases como container, row, y col-* para crear diseños.

jQuery: Biblioteca para facilitar JavaScript.
  Uso sencillo para manipulaciones del DOM.

WordPress: CMS para personalizar sitios sin necesidad de programación.
  Temas y plugins para añadir funcionalidades.


5. SEO (Optimización)

Etiquetas importantes:
 <title>: Título de la página.
 <meta description>: Breve descripción.

Prácticas recomendadas:
Usar URL amigables.
Imágenes con texto alternativo (alt).

6. Pruebas y Optimización
 Herramientas de prueba:
      Google PageSpeed Insights: Para verificar velocidad.
      W3C Validator: Para validar HTML/CSS.
Optimización:
  Comprimir imágenes.
  Minimizar CSS/JS.
